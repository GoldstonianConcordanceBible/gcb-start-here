# Goldstonian Concordance Bible Specification (GCB-SPEC)

This repository is the **specification (“constitution”)** for the Goldstonian Concordance Bible (GCB): a machine-readable computational theology framework built on the interpretive stack **Mirror → Water → Fire**.

## What’s in this repo
- **framework/**: the hermeneutic definition and rules
- **ontology/**: canonical theological entity definitions
- **schema/**: machine-readable schemas for annotations and doctrine
- **datasets/**: starter crosswalk and index datasets
- **agents/**: protocol for GCB-compatible agents (SydTek Scholars / SydTek University / NPCs)
- **curriculum/**: mapping rules for accreditation-ready alignment

## Mirror → Water → Fire (very short)
- **Mirror**: text + history (what it meant then)
- **Water**: covenant + doctrine (what it means across the canon)
- **Fire**: prophecy + fulfillment (where it is going)
